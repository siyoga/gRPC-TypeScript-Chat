## Setup chat

---

1. You need install all packages using:

```bash
npm install

# or

yarn install
```

2. Run build-protos.sh using:

```bash
sh build-protos.sh
```

3. Run client and server:

```bash
# run server:
npm start start-server
yarn start-server
# run client:
npm start start-client
yarn start-client *your name*
```
